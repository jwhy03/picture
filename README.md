# 6x6 세계 국기 메모리 게임

7살 아이와 아빠가 함께 즐길 수 있도록 만든 2인용 메모리 게임입니다.

## 실행 방법

아래 중 하나를 사용해 `index.html`을 브라우저에서 열어 실행하세요.

- 파일 탐색기에서 `index.html` 더블클릭
- 또는 터미널에서 간단 서버 실행:

```bash
python3 -m http.server 8000
```

그 후 브라우저에서 `http://localhost:8000` 접속.

## 규칙

- 보드는 6x6(총 36장) 카드입니다.
- 같은 국기 2장을 맞히면 1점 획득 + 같은 플레이어가 한 번 더 진행합니다.
- 틀리면 상대 차례로 넘어갑니다.
- 모든 카드를 맞히면 게임 종료, 점수가 높은 사람이 승리합니다.
